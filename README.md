# PokeAbility - Phase 3 - ZRP Interview  

## Overview 
This is the phase 3 project of ZRP interview. 

### Minimum Require 
The project got the minimum requirement of implement at least backend server that has an endpoint 
to retrieve a pokemon abilities using the pokeapi. [visit the pokeapi website: https://pokeapi.co/about]

Also is a minimum requirement that the abilities must be given sorted by name. 

### Example
Suppose we run the server at the port 3000 over the localhost, so we can use we 
can get the pokemon ability using: 
    
    curl -X GET "http://localhost:3000/api/pokemon/poke-abilities/{pokename}"
